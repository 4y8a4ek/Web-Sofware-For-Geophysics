# Web-Software-For-Geophysics
### Платформа для обработки трёхмерных геофизических данных и подготовки к математическому моделированию

# Описание
Проект предоставляет веб-интерфейс для загрузки бинарных воксельных данных (формат .raw) и их автоматической трансформации в поверхностную треугольную сетку (STL) с возможностью управления детализацией. Полученная сетка используется для построения топологических структур (вершины, рёбра, грани, элементы), необходимых для конечно-элементного моделирования физических процессов в геофизике: фильтрация, теплопроводность, акустика, упругость.

### Архитектура построена на микросервисном принципе с использованием очередей задач (Kafka), что обеспечивает масштабируемость и асинхронную обработку. Визуализация выполняется через Three.js, а метрики собираются в Prometheus и отображаются в Grafana.

# Ключевые возможности
1. Загрузка бинарных данных – поддержка .raw файлов (uint8, 1 байт на воксель).

2. Автоопределение размерности – размер сетки вычисляется как кубический корень из объёма файла.

3. Регулировка детализации – параметр «Фактор объединения» позволяет объединять соседние воксели (2×2×2, 3×3×3 и т.д.) для управления плотностью сетки.

4. Генерация STL – алгоритм Marching Cubes создаёт трёхмерную поверхность (твёрдая фаза).

5. Асинхронная очередь задач – все расчёты выполняются в фоновых воркерах через Kafka.

6. Построение топологической структуры – C# воркер преобразует STL в набор точек, рёбер, граней и конечных элементов с сохранением связей.

7. Веб-визуализация – интерактивный 3D-вьювер на Three.js с кастомным шейдером (градиент по высоте).

8. Мониторинг – встроенные метрики (количество задач, время обработки) и дашборды Grafana.

# Быстрый старт
1. Убедитесь, что установлены Docker и Docker Compose.

2. Клонируйте репозиторий:

```bash
git clone https://github.com/your-username/Web-Sofware-For-Geophysics.git
cd Web-Sofware-For-Geophysics
```
3. Запустите все сервисы:

```bash
docker-compose up --build
```
4. Откройте в браузере http://localhost:8000.
5. Загрузите .raw файл, укажите параметры (размеры и фактор) и нажмите «Построить».

6. После обработки модель отобразится в 3D-вьювере. Логи работы воркера доступны через:

```bash
docker logs -f worker
```
